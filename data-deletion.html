Permintaan Penghapusan Data

Terakhir diperbarui: 20 Juni 2026

MomenNaikSatu tidak menyimpan data pribadi pengguna. Aplikasi ini menggunakan API Meta hanya untuk keperluan internal — yaitu memposting konten ke akun Instagram milik pengelola channel.

Token akses (access token) yang digunakan untuk autentikasi API disimpan secara lokal di server pribadi dan tidak pernah dibagikan kepada pihak ketiga.

Cara Mencabut Akses Aplikasi

Jika Anda ingin mencabut akses aplikasi MomenNaikSatuApp dari akun Facebook Anda:

  1. Buka Facebook Settings → Apps and Websites
  2. Cari MomenNaikSatuApp
  3. Klik Remove

Setelah dicabut, semua akses token akan langsung tidak aktif dan tidak ada data yang tersimpan di sistem kami.
