Privacy Policy

Terakhir diperbarui: 25 Juni 2026

MomenNaikSatu ("kami", "aplikasi") menghormati privasi pengguna dan berkomitmen untuk melindungi informasi yang terkait dengan penggunaan layanan kami. Kebijakan Privasi ini menjelaskan data apa yang kami kumpulkan, bagaimana kami menggunakannya, berapa lama kami menyimpannya, dan hak-hak Anda sebagai pengguna.

1. Data yang Kami Kumpulkan

Aplikasi ini hanya digunakan oleh pengelola channel MomenNaikSatu. Data yang dikumpulkan melalui integrasi TikTok API meliputi:

  • Informasi profil dasar: open ID TikTok, nama tampilan, URL avatar — digunakan untuk memverifikasi akun yang diotorisasi.
  • Token akses OAuth: token yang dikeluarkan oleh TikTok setelah proses otorisasi — digunakan untuk mengunggah dan memposting konten atas nama akun yang diotorisasi.

Kami tidak mengumpulkan informasi pribadi dari penonton atau pengikut channel. Interaksi seperti like, komentar, dan follow dikelola sepenuhnya oleh platform TikTok sesuai kebijakan privasi mereka.

2. Cara Kami Menggunakan Data

Data yang dikumpulkan digunakan semata-mata untuk:

  • Mengotentikasi akun TikTok pengelola channel.
  • Mengunggah dan memposting video konten secara terjadwal menggunakan TikTok Content Posting API (video.upload, video.publish).

Data tidak digunakan untuk tujuan iklan, analitik pihak ketiga, atau dibagikan kepada siapapun selain pengelola aplikasi.

3. Penyimpanan dan Retensi Data

Token akses TikTok disimpan dalam variabel lingkungan terenkripsi pada server pribadi pengelola channel dan tidak pernah di-hardcode dalam kode sumber atau disimpan dalam database publik.

Periode retensi:

  • Token akses disimpan selama akun TikTok aktif diotorisasi ke aplikasi ini.
  • Token akan dihapus segera setelah pengguna mencabut otorisasi atau mengajukan permintaan penghapusan data.
  • Log aktivitas (waktu posting, ID video) disimpan maksimal 90 hari untuk keperluan debugging, setelah itu dihapus otomatis.

4. Hak Pengguna

Sebagai pengguna yang telah mengotorisasi aplikasi ini, Anda memiliki hak untuk:

  • Mengakses data: Meminta informasi tentang data apa saja yang kami simpan terkait akun Anda.
  • Menghapus data: Meminta penghapusan semua data terkait akun Anda. Lihat halaman penghapusan data atau hubungi kami di bawah.
  • Mencabut otorisasi: Anda dapat mencabut akses aplikasi ini kapan saja melalui pengaturan TikTok (Settings → Security → Apps and Websites). Pencabutan otorisasi akan menonaktifkan akses aplikasi secara instan.

5. Cookie dan Pelacakan

Situs web ini tidak menggunakan cookie, pixel tracking, atau teknologi pelacakan pihak ketiga apapun.

6. Keamanan Data

Kami menerapkan langkah-langkah keamanan berikut:

  • Token API disimpan sebagai variabel lingkungan terenkripsi, tidak pernah dalam plaintext.
  • Akses server dibatasi dan dilindungi dengan autentikasi SSH key.
  • Kode sumber tidak memuat credential apapun.

7. Perubahan Kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan signifikan akan dipublikasikan di halaman ini dengan tanggal pembaruan yang diperbarui.
