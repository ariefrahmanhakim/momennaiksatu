Kebijakan Privasi

Terakhir diperbarui: 19 Juni 2026

MomenNaikSatu ("kami") menghormati privasi pengguna dan berkomitmen untuk melindungi informasi pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengelola data ketika Anda berinteraksi dengan konten kami.

1. Informasi yang Kami Kumpulkan

Kami tidak mengumpulkan informasi pribadi secara langsung dari penonton. Interaksi dengan konten kami (like, komentar, subscribe) dikelola oleh platform masing-masing (TikTok, YouTube, Instagram) sesuai kebijakan privasi mereka.

2. Penggunaan API Platform

Kami menggunakan API resmi TikTok, YouTube, dan Instagram untuk memposting konten secara otomatis. Data akun creator (token akses) disimpan secara aman di server pribadi dan tidak dibagikan kepada pihak ketiga.

3. Cookie dan Pelacakan

Situs ini tidak menggunakan cookie atau teknologi pelacakan pihak ketiga.

4. Keamanan Data

Kami menggunakan langkah-langkah keamanan yang wajar untuk melindungi informasi. Token API disimpan dalam variabel lingkungan yang terenkripsi dan tidak pernah di-hardcode dalam kode sumber.

5. Perubahan Kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan akan dipublikasikan di halaman ini.
